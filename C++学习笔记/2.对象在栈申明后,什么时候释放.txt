对象在栈申明后，什么时候释放：
	对象申明后，在遇到“}”后释放，编译器不会判断后续有无使用而释放

1.
class A
{
public:
    A(){cout << "A" <<endl;}
    ~A(){cout << "~A" <<endl;}
};

int main()
{
    extern A a;
    cout << "1" <<endl;
    {
        cout << "2" <<endl;
        A a;
        cout << "3" <<endl;
    }
    cout << "4" <<endl;
    a;
    return 0;
}

输出:
1
2
A
3
~A
4
