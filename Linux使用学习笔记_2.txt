解压缩
tar -cjf all.tar.bz2 *.jpg 
这条命令是将所有.jpg的文件打成一个tar包，并且将其用bzip2压缩，生成一个 bzip2压缩过的包，包名为all.tar.bz2 
tar -xjf all.tar.bz2 
这条命令是将上面产生的包解开。

编译升级内核
<uname -r>
<yum install ncurses-devel>
<yum install make >
<yum install perl>
<cp 内核 /usr/src>
<tar -jxvf 内核>
<cd 内核>
<make mrproper> 第一次配置使用
<make clean>     以后或第二次配置使用的命令
<make menuconfig> 设置编译选项，选择本地文件系统语言为中文
<make dep> 设定依赖关系
<make bzImage> 压缩内核，当然也可以直接<make>，多核的话用<make -j 数字> 开多线程
<make modules> 编译内核模块
<make modules_install> 安装内核模块
<make install> 自动安装
